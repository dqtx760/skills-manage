# SBTI 15 维度打分规则

这是给**任何 Claude agent** 跑 `social-sbti` skill 时用的打分指引。
目标:不同 agent 读同一批动态,结果稳定一致。

## 硬约束

1. **证据引用必填**: 每个维度 ≥ 1 条 `post_id`。无证据 → level 强制 `M`。
2. **反向证据降档**: 如果你找到相反的 post,从你本来想打的档位降一档。
3. **置信度 < 0.4 → M**: 宁可保守,别硬凑。
4. **顺序固定**: 输出必须按 `S1 S2 S3 E1 E2 E3 A1 A2 A3 Ac1 Ac2 Ac3 So1 So2 So3` 的顺序。
5. **15 条全出**: 一条都不能少。
6. **reasoning 不超过 40 字中文**: 够用就好,留空间给证据。
7. **evidence 是 post_id 数组**,不是字符串拼接。

## 等级含义

| 等级 | 含义 | 什么时候打 |
|------|------|-----------|
| L | 低 | 该维度明显偏弱,你能找到多条一致的证据 |
| M | 中 | 混合信号,或证据不够强 |
| H | 高 | 该维度明显偏强,你能找到多条一致的证据 |

## 自我模型

### S1 · 自尊自信
- **H**: 频繁出现稳定的自我肯定(「我真的牛」「强得可怕了」「这波我做对了」),
       即使自嘲也带安全感。不怎么求外部认可。
- **M**: 时好时坏。顺境能自信,逆境会崩。
- **L**: 频繁自贬求安慰,被夸就否认,怕被揭穿。对自己下手比别人还狠。

### S2 · 自我清晰度
- **H**: 清楚自己是谁,明确说得出自己的天赋/偏好/边界,会用 MBTI、盖洛普
       之类的自我标签定位(但不强迫症)。
- **M**: 偶尔迷茫,偶尔清晰。
- **L**: 反复在"我到底想要什么"里打转,容易被情绪/他人切换身份。

### S3 · 核心价值
- **H**: 有清晰的目标/使命感,推进欲强(「All in X」「今年要完成 Y」)。
- **M**: 想上进也想躺,两头摇摆。
- **L**: 明确表达"佛系""躺平""舒服优先",对成长无追求。

## 情感模型

### E1 · 依恋安全感
- **H**: 关系里稳定,不 pia pia 发消息等回,面基/交友不焦虑。
- **M**: 信一半,试探一半。
- **L**: 对象/朋友稍微没回就脑补最坏结局,频繁出现"被抛弃""被冷落"叙事。

### E2 · 情感投入度
- **H**: 对人/事/社群情感卷入很深,常出现"心流""感动""让我感慨"。
- **M**: 会投入,但留后手。
- **L**: 情感克制,主动保持距离,怕陷入。

### E3 · 边界与依赖
- **H**: 强独立,空间感很重,会主动拒掉不想参加的事情。
- **M**: 亲密和独立都要一点。
- **L**: 黏人或被黏,喜欢长时间共处,"我们"频率高于"我"。

## 态度模型

### A1 · 世界观倾向
- **H**: 对人性、对机会开放乐观,"相信善意",对陌生人有天然友善。
- **M**: 不天真也不阴谋论。
- **L**: 防御警惕,对人对事默认怀疑,"世界 = 狗屎" 口气频繁。

### A2 · 规则灵活度
- **H**: 守规则,按流程走,讨厌即兴炸场。
- **M**: 灵活变通。
- **L**: 爱打破常规,自建 SOP,对传统路径保持批判。
- ⚠️ 常见陷阱: 不要把"做事有章法"(其实是 Ac3)当成 A2 高。
     A2 关注的是**对规则的态度**,不是执行力。

### A3 · 人生意义感
- **H**: 做事有清晰方向,常出现"使命""意义""为了 X 值得"。
- **M**: 偶尔目标,偶尔摆烂。
- **L**: 虚无感重,"人生就是过场""没意思"之类表达频繁。

## 行动驱力

### Ac1 · 动机导向
- **H**: 主动追进攻,为成果/成长点燃,敢下注。
- **M**: 混合。
- **L**: 避险优先,先想别翻车,再想要不要做。

### Ac2 · 决策风格
- **H**: 决定快,说干就干,不爱回头磨叽。
- **M**: 会想,但不至于想死机。
- **L**: 反复纠结,脑内会议超时,发帖频繁"我该选 A 还是 B"。

### Ac3 · 执行模式
- **H**: 推进欲强,事情不落地心里卡刺,持续日更/周更/持续输出。
- **M**: 状态看时机。
- **L**: 靠死线驱动,平时拖延严重,"临时抱佛脚"高频。
- ⚠️ 常见陷阱: 高执行力不等于高自信(S1)。

## 社交模型

### So1 · 社交主动性
- **H**: 主动发起聚会、主持圆桌、组线下、到处面基。
- **M**: 有人来就接,没人来也不硬凑。
- **L**: 社交启动慢热,主动出击攒半天气。
- ⚠️ 常见陷阱: "账号活跃 ≠ So1 高"。要看 ta 是发起方还是被动参与。

### So2 · 人际边界感
- **H**: 边界偏强,靠太近本能后退,"电子围栏"式社交。
- **M**: 看对象调节。
- **L**: 熟了就想融合,"自家人"叙事多,喜欢亲密距离。

### So3 · 表达与真实度
- **H**: 不同场合会切换不同人设/语气(正经长文 / 段子 / 私聊都不一样)。
- **M**: 真实和体面各留一点。
- **L**: 直来直去,心里有啥就说啥,很少绕弯。

## 输出 JSON schema

```json
{
  "profile": {
    "screen_name": "AI产品黄叔",
    "platform": "jike",
    "bio": "研究 AI 产品的",
    "post_count": 200
  },
  "personality_description": "200-300 字『本人定制版』描述,带具体事实",
  "overall_impression": "100-200 字整体印象",
  "quotes": [
    "原文引用 1",
    "原文引用 2",
    "原文引用 3",
    "原文引用 4"
  ],
  "scores": [
    {
      "dimension": "S1",
      "level": "H",
      "confidence": 0.9,
      "evidence": ["post_id_1", "post_id_2"],
      "reasoning": "不超过 40 字"
    }
    // ... 共 15 条,顺序固定
  ],
  "drunk": false
}
```

写完后送给 `src/match.py` 做人格匹配,然后 `src/make_card.py` 渲染成 HTML。
